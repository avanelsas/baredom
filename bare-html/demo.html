About BareDOM

BareDOM is a library of 56 native UI components built entirely on web standards — Custom Elements v1, Shadow DOM, and ES modules. No framework. No runtime. No virtual DOM.

Core rendering model 
DOM = f(attributes, properties)
56 Components
0 Runtime deps
100% Stateless

Every component compiles to a tree-shakeable ES module via Google Closure's advanced compilation pass. Import only what you use.
