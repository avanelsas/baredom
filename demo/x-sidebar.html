x-sidebar demo

Standalone demo page for the x-sidebar custom element. This page loads js/main.js, relies on app startup registration, demonstrates multiple states, and provides a live control panel for supported attributes and properties.

Live controls

Change attributes and properties at runtime. All emitted events are logged below and mirrored to the browser console.

Notes: collapsed only affects docked mode. Below the breakpoint, the effective variant becomes modal. Clicking items inside the sidebar does not auto-close by default; that behavior is consumer-controlled.
