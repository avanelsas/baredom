<x-proximity-list>

A list whose items scale up as the pointer approaches — the dock effect, applied to any list.

Description

Items are slotted as light-DOM children. The component listens for pointer movement over the flex track and writes transform plus three CSS custom properties (--x-proximity-influence, --x-proximity-scale, --x-proximity-lift) to each child. Items return to their base size via a CSS transition when the pointer leaves. Touch devices, reduced-motion, and disabled state all suppress the effect — click and keyboard activation still work.
