Drawer

This is the drawer body. You can place any content here.

Try pressing Tab to cycle focus within the drawer, or Escape to close it. Clicking the backdrop also closes the drawer.

The --x-drawer-size CSS custom property controls the width (for left/right) or height (for top/bottom).
