BareBuild — Read and Write Task demo (server state only)

A read and write projection of server state: the tasks live only on the server.

BareBuild is a small ClojureScript runtime that wraps the web components used on the page. It turns server responses into data the components can display, and turns user interactions back into fetch requests the server can handle.

Create and delete tasks, filter, sort a column, or change page. The query round-trips through the server into the URL. Invalid queries and network failures keep the last good view on screen.
