BareBuild — Task demo

A read and write projection of server state: the tasks live only on the server. BareBuild is a small ClojureScript runtime that wraps the web components on the page — it turns server responses into data the components can display, and turns user interactions back into requests the server can handle. Create and delete tasks, filter, sort a column, or change page; the query round-trips through the server into the URL. Invalid queries and network failures keep the last good view on screen.
